# Rahul Deshmukh | Cloud & Software Engineer Portfolio

A modern, responsive portfolio website built to showcase my expertise in cloud architecture, backend engineering, and AI systems.

## 🚀 Tech Stack
- **Frontend:** React, Vite, Tailwind CSS
- **Animations:** Framer Motion
- **Icons:** Lucide React
- **Deployment:** Github Pages

## ✨ Key Features
- **Dynamic Project Showcase:** Filterable projects by category (Cloud, AI, DevOps, Web).
- **Responsive Design:** Optimized for all screen sizes from mobile to desktop.
- **Glassmorphism UI:** Modern, high-performance aesthetic with subtle animations.
- **Project Modals:** Detailed views for each project with image galleries and architecture highlights.
- **Contact Obfuscation:** Built-in email protection against automated scrapers.

## 🛠️ Getting Started

1. **Clone the repo:**
   ```bash
   git clone https://github.com/Rahul-1611/Portfolio.git
   ```
2. **Install dependencies:**
   ```bash
   npm install
   ```
3. **Run locally:**
   ```bash
   npm run dev
   ```
